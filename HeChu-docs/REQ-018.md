## 需求描述
使用 ContentIsland 布局实现谜题界面，将谜题主体、元件、谜题信息三个区域清晰分离，独立显示。参考 WinUI-Gallery 的 ContentIslandPage.xaml 实现，提供更好的用户体验和界面组织。

## 边界条件
- 需要适配不同窗口大小和分辨率，确保在不同屏幕尺寸下都能良好显示
- 各区域之间的间距和比例需要合理调整，避免布局错乱
- 滚动和裁剪行为需要正确处理

## 技术路线
- 引入 ContentIsland 控件作为容器
- 将谜题界面划分为三个主要区域：
  1. 谜题主体区域 - 显示谜题网格和主要游戏区域
  2. 元件区域 - 显示可用的谜题元件（管道、节点等）
  3. 谜题信息区域 - 显示谜题的元数据、状态、提示等信息
- 使用 Grid 或 StackPanel 进行区域排列
- 添加适当的间距、边框和背景色以区分不同区域
- 确保响应式布局，窗口大小变化时自适应
